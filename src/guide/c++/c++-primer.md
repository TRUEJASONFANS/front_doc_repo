# 通用引用
具体来说，auto&& a = b 中的 a 是一个通用引用（Universal Reference），通用引用是由 Scott Meyers 在他的C++14中提出的有关引用折叠的概念。

通用引用的特点是，它可以引用不同种类的对象，无论是左值还是右值。这是通过引用折叠规则来实现的。根据引用折叠规则，auto&& 的类型会根据被引用对象的值类型来决定。

在 auto&& a = b 的语句中，如果 b 是一个左值，那么 auto&& 的类型将被推导为一个左值引用，相当于 auto&。因此，a 将成为 b 的左值引用。而如果 b 是一个右值，那么 auto&& 的类型将被推导为一个右值引用，相当于 auto&& 本身。因此，a 将成为 b 的右值引用。
